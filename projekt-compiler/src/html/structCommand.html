Detailed Description

Represents an Rail command, consits of an enum Command::Type and an argument. For each command type in Rail there exists a corresponding enum Command::Type.

Member Enumeration Documentation

enum Command::Type

Represents an command type of an Rail progamm. Each value corresponds to the ascii value of the corresponding Rail command. E.g. o -> 111 or a -> 97, thats makes the mapping much easier.
