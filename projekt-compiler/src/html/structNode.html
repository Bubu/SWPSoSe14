Detailed Description

Represents a node in the [a]bstract [s]yntax [g]raph (ASG), each node also represents a command in the corresponding Rail programm. That means each node has a command to execute.

std::shared_ptr<Node> Node::successor1

The first successor of the current node. This successor is always the true path and is set if a successor exists. That means if the node (command) is not #, what means end, a successor must exists and these is placed in the successor1.

std::shared_ptr<Node> Node::successor2

The second successor of the current node. This successor is only not null if the current node is a junction, like <, >, V or ^. The successor2 contains the false path of the junction.
